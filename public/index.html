Chitkara BFHL Challenge

Build hierarchies from node strings.

Paste a JSON array or one edge per line, submit to the hosted /bfhl endpoint, and inspect the parsed trees, invalid entries, duplicate edges, and summary counts in a compact dashboard.

Input Node List

Examples: ["A->B", "A->C", "B->D"] or one entry per line.

The API is called with POST /bfhl using JSON. CORS is enabled on the server.

API Response

Structured view of the hierarchies and metadata returned by the API.

Tip: paste the full validation payload from the prompt to confirm duplicate handling, cycle detection, and root selection.
